# Reinforcement-Learning-Mario

Se recomienda encarecidamente utilizar [Python 3.10](https://www.python.org/downloads/release/python-3100/) ya que versiones mas nueva puede implicar problemas.

play.py -> probar alguno de los checkpoints en train/checkpoints. 

visualtrain.py -> correr un entrenamiento visual del modelo usando PPO.

fine_tune.py -> ajustes de hiperparámetros tras la la pérdida de entropía y catastrophic forgetting.

# Crear el entorno
```
python -m venv venv
```
# Activar el entorno
```
.\venv\Scripts\activate
```
# Instalar dependencias
```
cd mario-rl
pip install -r requirements.txt
```
